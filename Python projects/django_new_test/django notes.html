Remember!

  1. To start a virtual environment for django do 'python -m pipenv shell'
  2. Then do 'pip install django'
  3. Then do 'django-admin startproject (name of project)'
  4. 'cd' into your (name of project)
  5. Then do 'django-admin startapp (name of app (9/10 times make it 'main'))
  6. 'python manage.py runserver' to get it running
  7. fn + ctrl + c to stop the server running
